# Roll a Dice Game 

** A simple JavaScript game where players role a dice. Players accrue points each round and 30 points are required to win the game**

## Features
1. Players name customisation:
    - Players can customise their names when they join the game.
2. Add multiple players:
    - New players can be added to the game and their name will be added to the scoreboard.
3. Dice roll button:
    - Each player has their own dice. Clicking the dice will generate a random number and add it to the score.
4. Real time scoreboard:
    - The scoreboard updates dynamically after each round to reflect the current points of all the players.
5. Edit name:
    - Players can edit their name at any time by clicking the "Edit Names" button.
6. Reset Score:
    - Players can reset the game scores to zero by clicking the "Reset Score" button. 

## How to start the game
1. Clone repo
2. Open index.html in your favourite browser
3. To reset game refresh page

## How to Play
1. Add Players:
    - Players can enter their name and click the "Edit Names" button to join the game.
2. Roll the Dice:
    - Each player rolls the dice. A random number between 1 and 6 will be generated for each roll, and the points are added to the player's total score for that round. 
3. Scoreboard:
    - The game features a live scoreboard that shows all players and their current points. 
4. Win Condition:
    - The first player to reach 30 points wins the game. When a player wins, the game will notify them.
