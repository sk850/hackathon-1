## Roll a Dice Game 

** A simple JavaScript game where players role a dice. Players accrue points each round and 30 points are required to win the game.nts first to win the game**

#Features
1. Players can change names
2. Players can be added to the game
3. Players click a button to roll the dice. Each player has 1 dice.
4. A scoreboard will be available for all players to view. Players are added to the scoreboard and the points earned refresh each round.
